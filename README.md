# SmartSpends AI — Розумний фінансовий трекер

**SmartSpends AI** — це веб-додаток на базі штучного інтелекту (Google Gemini), який допомагає користувачам легко відстежувати свої витрати, використовуючи звичайну людську мову замість складних таблиць.

---

## Основні функції

* **Додавання витрат текстом:** Просто напишіть "Купив каву за 60 грн" — ШІ сам визначить категорію та суму.
* **Інтелектуальний аналіз:** Отримуйте поради щодо економії на основі ваших реальних витрат.
* **Візуалізація даних:** Інтерактивні графіки та діаграми (Plotly) для наочного аналізу бюджету.
* **Автоматична категоризація:** Система автоматично групує витрати за категоріями (Їжа, Транспорт, Одяг тощо).

---

## Технологічний стек

* **Мова:** Python 3.10+
* **Інтерфейс:** Streamlit
* **База даних:** SQLite (локальна легка БД)
* **Штучний інтелект:** Google Generative AI (Gemini Pro API)
* **Аналіз даних:** Pandas, Plotly
